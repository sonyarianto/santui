# What is Santui?

Santui is **your terminal home base** — a lightweight, keyboard-driven TUI app that lives in your terminal. Think of it as a **launcher for plugins**: the core is minimal, and everything extra comes through the Plugin Registry.

It's not a framework or library for building other apps. It's the app itself, designed to be:

- **Extensible** — plugins run as separate processes over JSON IPC. Install them from the registry, enable them, and they appear in your command palette.
- **Themeable** — 38 OpenCode themes you can switch at runtime with live preview.
- **Keyboard-driven** — everything is a key press away, from the command palette to navigation.

## Built-in features

- **Command Palette** — `Ctrl+P` to search and execute commands instantly
- **Plugin Registry** — browse, install, enable, and disable plugins without leaving the terminal
- **Themes** — 38 built-in themes with instant switching and live preview
- **Auth** — sign in with Google or GitHub
