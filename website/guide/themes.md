## Color Mapping

The OpenCode palette keys map to Santui's `Theme` struct as follows:

| OpenCode | Santui | Usage |
|----------|--------|-------|
| `neutral` | `background_panel` | Dialog and panel backgrounds |
| `ink` | `text` | Primary text color |
| `primary` | `highlight`, `logo`, `border` | Highlight bar, logo, borders |
| `accent` | `accent` | Accent text, active item markers |
| `success` | `success` | Success indicators |
| `error` | `error` | Error indicators |
| `neutral` + `ink` | `text_muted` | Muted text (60/40 blend) |

## Switching Themes

Press `Ctrl+P` to open the command palette, then select **Switch theme**. A searchable theme picker appears — use ↑↓ to browse (with live preview), type to filter, `Enter` to select, `Esc` to cancel.
